# World Cup 2026 Predictions League

A private predictions game for the FIFA World Cup 2026 group stage. Next.js +
TypeScript + Tailwind, deployed on Vercel, data + auth on Supabase.

- **Read `CLAUDE.md` first** — it's the project bible (rules, schema, phases, changelog).
- **Follow `SETUP.md`** to stand up the database and deploy.

Built in three phases: (1) foundation + manual setup, (2) admin panel + seed data +
scoring, (3) prediction pages + leaderboards. Currently: **Phase 1**.
